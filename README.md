# Single-DC-OISM-Demo
This is a containerlab topology that can be used to demo/test OISM in a single DC, in addition to PEG and PEG transit features.
This topology is based on the topology from the following webinars:

[5/13/20 Webinar: Arista Technology Series Designing and Deploying EVPN VxLAN Networks](https://www.youtube.com/watch?v=Wq5CyihsLwo)
[5/27/2021 Netbuilders EVPN Series - Session 8: EVPN OISM Deep Dive](https://www.youtube.com/watch?v=GfL76fBvAxA)
[3/20/2025 Netbuilders EVPN Series Session 9: Advanced deployment models with EVPN OISM (Part 3)](https://www.youtube.com/watch?v=Cc3EhNAn3iE)

It also takes some configuration from the following github repo: [PacketAnglers oism-demo Github Repo](https://github.com/PacketAnglers/oism-demo/tree/main)

It is set up to use ceos 4.35.0F.
I have also used the VS Code Containerlab Extension to generate a topology file for it.
